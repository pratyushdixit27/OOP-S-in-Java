      Introduction to Object Oriented Programming System(OOP'S) in JAVA

> What is OOP's?
> It stands for Object Oriented Programming System/Structure. It is a programming paradigm or methodology.

Paradigm is a method to solve some program or to do some tasks.
Their are six pillars in OOPS:
1) Class
2) Objects/Method
3) Inheritance
4) Abstraction
5) Polymorphism 
6) Encapsulation
